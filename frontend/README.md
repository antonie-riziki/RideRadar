# RideRadar Frontend

This folder contains a React (Vite) frontend scaffold for the RideRadar project.

Quick start (frontend dev):

```bash
cd frontend
npm install
npm run dev
```

Run backend concurrently (from project root):

```bash
python3 manage.py runserver
# or run Django in parallel and use the React dev server
```

Integration notes are in `docs/frontend_integration.md`.
